# OG Image

为了更好的社交媒体分享效果，请在 `public` 目录下添加 `og-image.jpg` 文件。

## 建议尺寸
- **尺寸**: 1200x630 像素
- **格式**: JPG 或 PNG
- **大小**: < 8MB（建议 < 300KB）
- **内容**: 包含网站标题、简短描述或代表性的 AI 艺术作品

## 图标文件
同时建议添加以下图标文件：
- `icon-192.png` (192x192)
- `icon-512.png` (512x512)
- `favicon.ico` (32x32)

这些文件将用于：
- PWA (Progressive Web App) 支持
- 移动设备主屏幕图标
- 浏览器标签图标
